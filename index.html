Introduction

Python is a high-level programming language that was created by Guido Van Rossum in the late 1980s. The language priorities readability and simplicity. It is and can be used for several purposes such as desktop application, mobile application, machine learning and games.

This is good languages for anyone who wants to learn programming and does not come from a programming background. According to many programmers, it is the easiest to learn. Python takes less lines of code which would take many lines in other programming languages. Following is an example of printing something to screen using java and python.

  • Python  public class Main {
      public static void main(String[] args) {
       System.out.println("Hello World");
      }
     }
  • Java print("Hello World")

Installing

Unlike HTML, CSS, and JavaScript, python programs cannot be executed out of the box. Some programs need to be installed before executing a python program. To be specific, python interpreter needs to be installed. It can be downloaded from the python website. Installing the latest version is preferable.

In order to write the python programs a code editor is needed. It can be anything from the notepad that comes with the operating system (Windows or Mac) or a text editor that is specifically designed for programming such as Visual Studio Code. Python interpreter also comes with a code editor. Additionally, there is python shell in which python commands or code can be tested. The shell also comes with the interpreter, and it can be useful tool when learning to program as a beginner. Following are some examples of code editors.

  • Atom
  • Sublime Text
  • Brackets
  • Vim
  • Visual Studio Code

Variables

A variable is a data that is stored under a name which can later be manipulate using its given name. Following code explains how to define a variable

name = "John Doe"

‘name’ is the variable name and ‘John Doe’ is the data or value assigned to it.

The equal sign (=) is called assignment operator. It assigns the value on the right to the variable on the left.

Following are some rules for naming variables.

  • Variable names can only contain letters, numbers and underscore.
  • Names cannot start with a number.
  • Reserve words such as ‘print’ cannot be used.
  • Variable names are case sensitive

Data

There are many different types of data are built into python and other programming languages. However, this documentation covers only 3 of the following.

  • Integer

    • Integer refers to positive and negative whole numbers. They have no decimals and no limit to their length.
    • Ex: -5, 0, 3

  • Float

    • Float numbers contain decimal points, and they can be positive or negative.
    • Ex: -2.3, 8.3

  • String

    • It is a text within single or double quotes.
    • Ex: “Hello World”, “45”, ‘John’.
    • String has several built-in functions.
